multi_scale_NIF.ipynb: 
	Jupyter notebook code for nif model, including training and testing process

Folder:
	Data: including training data and testing data, if we want to use the data, 
	firstly we need to get the data from matlab, and then use python to convert it
	to the .npz format.

Folder:
	saved_weights_multiscale: the saved model parameters of the trained nif model.
